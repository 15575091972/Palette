# Palette
前后端分离的艺术品交易平台
本系统以以太坊(Ethereum)的 Geth 客户端平台搭建的区块链为核心，搭配 MySQL关系型数据库，独立开发的集合网页和 Android 为一体的数字艺术品交易平台。
本系统开发的目的是，通过该平台解决现实中存在的艺术品流通有着难以验证真伪、受众面小、难以分开销售（比如图库）等痛点，实现艺术品数字化上链及交易功能。
